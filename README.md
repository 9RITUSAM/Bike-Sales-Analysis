# Bike Sales Analysis Dashboard

An interactive Excel dashboard built to analyze bike sales data and understand customer purchasing behavior. The dashboard transforms raw data into clear visual insights, helping identify trends and patterns that influence bike purchases.

## Project Overview
This project focuses on exploring customer demographics and purchasing trends using Excel's data analysis and visualization features. The goal is to present insights in a simple and interactive format that supports quick decision-making.

## Key Features
- Interactive dashboard created in Microsoft Excel
- Data analysis using **Pivot Tables**
- Visualizations with **Charts and Graphs**
- Interactive filtering using **Slicers**
- Analysis based on customer attributes such as:
  - Age
  - Income
  - Occupation
  - Commute Distance

## Insights Generated
- Identified customer segments more likely to purchase bikes
- Analyzed how income and commute distance influence purchasing decisions
- Visualized demographic patterns affecting sales trends

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Slicers for interactive filtering

## Outcome
The dashboard presents sales insights in a clear and structured format, enabling quick analysis of customer behavior and supporting data-driven decision making.
